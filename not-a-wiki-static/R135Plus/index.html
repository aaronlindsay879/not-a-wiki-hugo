Angelline Demons (R139+) 1e95 (100 Tg)+

Author: Rellikrellik, Wolfsblood

Requirement: Djinn Challenge 2, Angel Lineage Lineage 40, 3e7 (30 M)+ Hellfire Blast casts this Reincarnation

Range: 1e95 (100 Tg) Gems+

Faction: Demon

Bloodline: Angel

Artifact Set: Demon

2e8 (200 M) HFB casts this R

R154+, e140 (100 Qiqag) Gems+

S435,S3200,

C50,C340,

D10,D560,D1375,

E230,E320,E495,

A25,A400,A410,

W1,W135,W175,W405,W520,W590

Note: This build requires a buffed W175 to quickly afford Djinn Treaty, run the "Early W175" buff build once until you have enough Hellfire Blast casts.

Note: Swap E320 with E135 once you already have 2e8 (200 M) Hellfire Blast casts this Reincarnation.

Note: R154+, e140 (100 Qiqag) Gems+ & long runs - swap E320 with E400.

Note: A quick Spell casts buff (Spell buff build for 5 to 10 minutes) is helpful, more is not essential.

Note: While Dragonline Angel can still push to e100 gems it is more optimal to start with Demon runs at around e95 gems.

Note: This build starts slow but has insane growth thanks to the feedback loop between Hellfire Blast, Limited Wish and DJC2.

Note: The build's production is focused around Non-Unique Buildings; because of DJ8 (based on UTC) - the main production tier is either Evil Fortress (even calendar days) or Hell Portals (odd calendar days - do not buy Infernal Realm upgrade in that case).

Note: Set Drow Chaos Union to target that Main Production Tier.

Note: Build can be run idle, but needs to be online, ideally doing manual setup once or twice at the end before abdicating.

Note: First Run Length: 1.7e8 Hellfire Blast casts.

Note: Consecutive Run Length: Relation between Hellfire Blast casts this game / this R should be at least 60% (rule of thumb).


Optimal Manual Setup:

- Maelstrom set to main production tier (Evil Fortresses or Hell Portals) based on trophies.

- Limited Wish -> trophy count (LW's strength is also random, check Realm Weather Service in Advance - should be a 90% or better hit).

- Catalyst set to Gem Grinder.

- (Djinn Challenge 4 gives 888% Assistants)

Order of upgrades for Manual Setup:

- Wait for DJC4, activate Catalyst, activate Maelstrom, and finally when all spells run, Limited wish.
